# SH Atelier AI Tools

ابزارهای هوش مصنوعی برای کمک به خیاط‌ها، طراحان لباس و تولیدکنندگان پوشاک.
این پروژه متن‌باز است و با هدف ساده‌سازی فرآیند طراحی، دوخت و تولید محتوا ساخته شده.

## امکانات
- تولید کپشن حرفه‌ای برای پست‌های اینستاگرام برندهای پوشاک
- تبدیل عکس لباس به توضیحات فنی قابل استفاده در کاتالوگ
- پیشنهاد اصلاحات دوخت و نکات Tailoring
- ساخت جدول سایزبندی استاندارد برای لباس‌ها
